  • Simulation-Chip Test Mode

    Edit program freely, replaceable chip

    Whether you are developing a prototype of a product or creating a standard test method, you can use our simulation function.You can edit electrochemical scanning method,detection duration,enrichment time, correction curve freely and then burn these to the chip, you can use touch panel to control the test without computer, and read the results directly on the screen.

    • It only takes 4 steps to complete process editing
      (Procedure / Data analysis / Concentration transfer / Action to burn-in)

    • Each ECA methods can be edited interchangeably
      (CV / IT / EIS-AC Impendance)

    • Match up the sample injection status for interchangeable editing
      (Wating for sample / Wating for second / Wating for trigger button)
