Full-Function Variable Frequency Magnetic Stirrer for Electrochemical Experiments
(Electrochemical Cell stand)

CS100

  • Different solution viscosity, same stirring rate
  • Temperature control
  • countdown function
  • Electrochemical experiment special holder

  • This is a Magnetic Stirrer specifically designed for electrochemical experiment. In addition to providing constant stirring rate control, the variable frequency magnetic stirrer also equips with additional functions such as constant temperature control and countdown timer and its uniquely designed holder for electrochemical experiments allows you to obtain a stable working platform whether using three-electrode system or using screen printed electrodes.

  • Different solution viscosity, same stirring rate

    Automatically responsive to different viscosity, constant-speed magnetic stirrer

    Automatically responsive to different viscosity, constant-speed magnetic stirrer Did you know that due to different viscosity of the solution, the same setting of general magnetic stirrer may get different stirring rate because the rate setting of the non-variable frequency magnetic stirrer corresponds to the power output of stirring pure water at this rate? When the solution becomes thicker, the actual stirring speed will be lower than the set value, but this variable frequency magnetic stirrer does not have this problem. It is suitable for almost all liquid such as ionic liquid, colloid, etc. in electrochemistry lab and it will provide steady stirring speed as you need.

  • Constant-temperature heating system with countdown function

    Provide 20~40 +- 2℃ constant temperature control system, and equipped with a safe countdown function

  • Electrochemical experiment special holder

    Multiple fixing modes are provided

    The reference electrode is preferred to be constantly placed as close as a distance of 2d to the working electrode (d: tip diameter of reference capillary) to avoid shielding problem or to minimize the potential drop caused by the cell resistance between WE and CE (iRcell) Electrochemical Methods and Applications 2nd, 2001 (Allen J. Bard, Larry R. Faulkner). So using electrochemical experiment special holder or three-electrode SPE can provide a stable electrochemical system for experiments and reduce system error.
